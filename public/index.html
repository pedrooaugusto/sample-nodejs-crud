CRUD com NodeJS

Demonstração de uma aplicação de CRUD usando Nodejs em conjunto com o banco de dados MySQL.

A aplicação consiste na manipulação da tabela:

entregador
* ID_ent int
Nome_ent varchar(45)
Salario_ent float
DataNasc_ent varchar(10)
CPF_ent varchar(20)
Veiculo_ent varchar(10)

Construido com:

  • Nodejs - como linguagem no servidor
  • MySQL - como banco de dados
  • Express - como servidor
  • Bootstrap - como front end framework
